Detailed Description

template<typename T>
class Matrix< T >

Template class representing a mathematical matrix.

Provides basic matrix operations such as addition, subtraction, multiplication (both scalar and matrix), transpose, determinant calculation, and methods for generating special matrices (identity, random).

Template Parameters
TThe type of the elements stored in the matrix.
